# 智能旅行助手平台

为用户提供旅行规划与图像记录服务。支持多轮对话、知识库检索、工具调用等能力，智能生成个性化行程并导出 PDF。平台同时集成图库管理功能，分为个人图库、共享图库和团队图库三大模块，支持旅途照片上传、批量管理、AI图片编辑、多维检索与实时协同编辑，满足个人、情侣和团队在旅行过程中的规划与记录需求。

---

## ✨ 核心功能

### 🧠 旅行规划智能助手

- 基于 ChatMemoryAdvisor 实现多轮对话记忆，结合 Kryo 序列化持久化上下文，保障语境连贯并提升系统稳定性。
- 通过文档切片、语义向量存储、多维过滤与查询增强，并集成阿里云百炼知识库，实现高效精准的 RAG 检索。
- 利用 Spring Al 的工具调用注解，支持文件处理、网页抓取、搜索下载及 PDF 生成等多种操作，扩展了 AI 的能力边界。
- 构建 ReAct 模式 AI 智能体，具备任务分解、自主决策、工具选择与循环执行等自主规划与复杂任务执行能力。

### 🖼️ 图像记录服务

平台集成强大的图像管理模块，分为三大图库：

- 个人图库：用户可上传、批量管理旅途照片，支持 AI 扩图、加水印、智能标签与颜色搜索。
- 共享图库：支持上传公开照片素材，通过文件或 URL 导入，平台审核后对外展示。
- 团队图库：面向家庭、情侣、企业旅行团体，支持空间共享、成员协同管理与实时图像编辑协作。

图库系统支持可视化图片分析，实现旅行图像的结构化处理与展示。

## 🛠 技术架构

| 模块 | 技术栈 |
|------|--------|
| 后端框架 | Spring Boot、Spring AI |
| AI 能力 | RAG、Tool Calling、多轮对话、大模型接入（通义千问、Ollama） |
| 服务集成 | MCP 服务总线 |
| 数据存储 | MySQL、Redis、COS（对象存储） |
| 缓存策略 | Redis + Caffeine 多级缓存 |
| 权限管理 | Sa-Token 权限控制 |
| 实时通讯 | WebSocket |
| 并发优化 | JUC、Disruptor |
| 图像处理 | AI 扩图、加水印、颜色识别、多维检索 |
